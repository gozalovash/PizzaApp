# PizzaApp
Semester Project - "PizzaApp" Desktop Application, Information Security faculty

Prerequisites:
- Python3
- OOP
- Tkinter
- Sqlite

Content:
- Main.py - the starting point, login and register windows;
- Admin_page - page, to which admin is redirected after login;
- Order_Pizza.py - page where the user can choose, design and add Pizza to the Cart;
- Pizza_Checkout.py -  checkout page, to which user is redirected after choosing Pizza;
- Pizza.py - script, where Pizza decorators  and Pizza Builder is located;
- Generate_db.py - class is called to generate database tables

Future plans:
- Let admin construct new Pizza and add it to user menu
